# Перевірка граматики тексту (OpenAI)

Цей проєкт передбачає створення веб-кабінету, у якому користувач може
вставити текст, перевірити його граматику за допомогою OpenAI та
переглядати історію перевірок у своєму кабінеті.

## Використані технології
- OpenAI API
- JavaScript / TypeScript
- React
- Node.js
- PostgreSQL
